# Task 01 # Task 01 – Cybersecurity Fundamentals

## Internship
Elevate Labs – Cybersecurity Internship

## Task Overview
This task focuses on understanding the fundamental concepts of cybersecurity, including:
- CIA Triad (Confidentiality, Integrity, Availability)
- Types of cyber threats and threat actors
- Common attack surfaces in modern systems
- OWASP Top 10 (2025)
- Mapping daily-used applications to attack surfaces
- Data flow between user, application server, and database

## Deliverables
- **Task-1_Cybersecurity-Fundamentals.pdf** – Detailed explanation of concepts
- **Data-Flow-Diagram.png** – Visual representation of data flow and attack points
- **Attack-Surface-Mindmap.png** – Mapping attack surfaces for daily-used applications

## Outcome
This task helped build a strong foundational understanding of cybersecurity concepts and how real-world systems are exposed to threats and attacks.
